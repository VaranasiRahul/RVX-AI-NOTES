## 📖 About

**RVX AI Notes** is a mobile application that helps you passively revise your notes throughout the day. It ingests text-based notes, runs on-device AI summarization (no internet required), and surfaces them in an intelligent, algorithmic feed — similar to how social media works, but for your own knowledge.

Designed, developed, and owned entirely by **Rahul Varanasi** — from concept to production build.

---

## ✨ Features

| Feature | Description |
|---|---|
| 📂 **Folder-based Notes** | Organize notes into topic folders |
| 🤖 **On-device AI Summarization** | Summarizes blocks using a local ML model (ONNX Runtime) — no cloud, no data sharing |
| 📰 **Intelligent Feed** | Algorithmic card-based feed that surfaces notes for revision |
| 🔖 **Bookmark System** | Save important blocks with a single tap |
| 📖 **Story-style Review** | Swipe through bookmarked notes in a distraction-free story view |
| 🪟 **Android Home Widget** | Glanceable widget showing your bookmarked revision blocks |
| 🌙 **Dark Mode** | Full dark mode UI throughout |
| 📴 **100% Offline** | All AI and data processing happens on-device |

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| **Framework** | React Native + Expo (SDK 52) |
| **Language** | TypeScript |
| **Local AI** | ONNX Runtime (React Native) |
| **Database** | SQLite via Drizzle ORM |
| **Navigation** | Expo Router (file-based) |
| **Android Widget** | react-native-android-widget |
| **Build** | EAS Build (Expo Application Services) |
