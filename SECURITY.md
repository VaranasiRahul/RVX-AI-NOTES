# Security Policy

## Supported Versions

Since this is a proprietary, closed-source project intended solely for portfolio and hiring review purposes, no public versions are officially supported for external use.

## Reporting a Vulnerability

If you form part of an invited review or hiring process and discover a potential security vulnerability within this repository, please respect the proprietary nature of this codebase.

**Do NOT file a public issue.**

Instead, please report the issue privately by emailing the owner directly:

- **Email:** rahulvaranasi04@gmail.com
- **Subject:** Security Vulnerability Report: RVX AI Notes

Please include:
- A description of the vulnerability
- Steps to reproduce it
- (Optional) Suggested mitigation

I will review the report and respond as soon as possible. Thank you for your cooperation and respect for the IP of this project.
